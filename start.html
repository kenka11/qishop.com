Шаг 3

Установите TLauncher

Запустите скачанный файл установщика, он установит мультиплеер

Шаг 4

Запустите TLauncher

1. После установки запустите TLauncher

2. Впишите свой ник в поле Логин, после выберите версию 1.13+

3. В открывшемся окне перейдите в "Сетевая игра" Дальше добавьте наш сервер (IP адрес нашего сервера: play.YOURSERVER.ru)

4. Нажмите "Подключится". Подтвердите скачивание ресурс - пака и наслаждайтесь игрой!
